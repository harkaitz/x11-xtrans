# XTRANS

## Dependencies.

- trans
- thingmenu
- xsel,xclip
- locale
- xdotool

## Help

xtrans

    Usage: xtrans LANGS
    
    This program shows a little button, when clicked it puts the
    translation of the selection in the clipboard.
